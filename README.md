Skill Exchange Platform 

A peer-to-peer learning and skill-sharing platform developed as a Final Year Project (FYP). The platform connects learners and skilled individuals, allowing users to teach, learn, and exchange knowledge through an interactive and collaborative environment.

About

Skill Exchange Platform aims to bridge the gap between learners and experts by creating a community where users can share their skills, offer mentorship, and learn new competencies. The platform encourages knowledge sharing, collaboration, and continuous personal development.

Modules
User Management

User registration, login, profile management, and secure authentication for learners and mentors.

Skill Marketplace

Users can create, browse, and search skill listings based on categories such as Programming, Design, Languages, Business, and more.

Learning & Teaching System

Allows users to offer skills, request learning opportunities, and connect with suitable mentors or learners.

AI Recommendation Module

Provides personalized skill and mentor recommendations based on user interests, learning history, and preferences.

Communication Module

Enables interaction between users through messaging and collaboration features.

Rating & Feedback System

Users can rate learning experiences and provide feedback to maintain quality and trust within the platform.

Admin Dashboard

Administrative panel for managing users, skills, reports, platform content, and overall system operations.

Tech Stack
Flutter (Frontend Mobile Application)
Node.js & Express.js (Backend API)
PostgreSQL / MySQL (Database)
Firebase Authentication
Cloudinary (Media Storage)
RESTful APIs
AI Recommendation Module
Access Model

The platform follows a user-centric model where learners and mentors can create accounts, manage profiles, exchange skills, and collaborate through a secure and interactive environment.
